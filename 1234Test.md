xyz = 23.
\end{cases}
\]
Using the assumption \(x = y\), we get \(x = 2\) and \(z = \frac{23}{4}\). This yields
\[
x^2 + y^2 + z^2 = 2(2^2) + \left(\frac{23}{4}\right)^2 = 8 + \frac{529}{16} = \frac{657}{16}.
\]
Hence, the square of the diagonal is \(d^2 = \frac{657}{16}\), so the smallest enclosing sphere has radius \(r = \frac{d}{2}\). Thus,
\[

\]
Therefore, \(p + q = 657 + 64 = 721.\)
